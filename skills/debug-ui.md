# Skill: Debug UI Issue

## When to use
- UI not rendering properly
- Layout issues
- Unexpected visual behavior

## Steps
1. Check component structure
2. Verify props and state
3. Review styling (CSS / layout)
4. Identify minimal root cause
5. Fix only the issue without rewriting entire component

## Rules
- Do NOT rewrite whole components
- Do NOT introduce new libraries
- Prefer minimal fixes
